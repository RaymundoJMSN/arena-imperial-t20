## Política de Privacidade — Arena Imperial T20

### Coleta de dados

**Arena Imperial T20 não coleta, armazena ou transmite nenhum dado pessoal.**

Toda a informação que você insere na ferramenta (grupos de personagens, encontros salvos, filtros, preferências) é armazenada **exclusivamente no seu navegador** via `localStorage`. Esses dados nunca saem do seu dispositivo.

### Cookies e rastreamento

Esta ferramenta não utiliza cookies, analytics, rastreamento de terceiros nem qualquer tipo de telemetria.

### Dados de monstros

O banco de dados de monstros é carregado localmente a partir de arquivos incluídos na aplicação. Nenhuma requisição é feita a servidores externos para buscar dados de criaturas.

### Links externos

Esta ferramenta contém links para o site da Jambô Editora e outras páginas externas. Ao acessar esses links, você está sujeito às políticas de privacidade desses respectivos sites.
